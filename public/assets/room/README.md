Room Art Drop Folder

Place open-source room art files in this folder. The app will auto-load these filenames:

Base layers
- `wall.png` (optional)
- `floor.png` (optional)
- `furniture.png` (optional)
- `bookshelf.png` (optional)
- `atmosphere.png` (optional)

Dark mode (optional)
- `wall-dark.png`
- `floor-dark.png`
- `atmosphere-dark.png`

Notes
- WebP with transparent backgrounds works best for decor sprites.
- Recommended canvas size: around `1920x1080` for base layers.
- Keep asset licenses in your own records (source URL, license type, attribution requirements).
